# Yarov_dz_oop — Калькулятор римских чисел

Домашнее задание по дисциплине «Объектно-ориентированное программирование».

## Тема
**Римские цифры** — Сложение, вычитание, умножение, целочисленное деление, остаток от целочисленного деления, перевод из римской системы счисления в десятичную и обратно, сброс результата.

## Структура проекта

```
Yarov_dz_oop/
├── RomanNumber.cs          # Основной класс — римские числа с перегрузкой операторов
├── MainForm.cs             # WinForms GUI калькулятора
├── MainForm.Designer.cs    # Designer-заглушка
├── Program.cs              # Точка входа
├── RomanNumberTests.cs     # Тесты (конвертация, арифметика, исключения)
└── Yarov_dz_oop.csproj     # Файл проекта (.NET 10, WinForms)
```

## Функциональность

- ✅ Класс `RomanNumber` с инкапсуляцией, перегрузкой операторов `+`, `-`, `*`, `/`, `%`, `==`, `!=`, `<`, `>`, `<=`, `>=`
- ✅ Конвертация: десятичная → римская и римская → десятичная
- ✅ Графический интерфейс (WinForms) с историей операций
- ✅ Обработка исключений (диапазон, некорректный ввод, деление на 0, отрицательный результат)
- ✅ Тестирование (30+ тестов)

## Запуск

```bash
cd Yarov_dz_oop
dotnet run
```
